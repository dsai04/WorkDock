WorkDock – Online Job Portal

Project Overview

WorkDock is a full-stack online job portal designed to manage job postings and applications efficiently. The platform enables companies to post job listings and allows users to browse, apply, and manage applications through a structured database system.
The primary objective of this project was to design a scalable backend architecture and implement structured database queries to handle job and user data effectively.

🛠 Technologies Used
  
    HTML

    Python

    SQL

    MySQL

🔧 Key Features

    User registration and authentication

    Company job posting functionality

    Structured relational database design

    SQL-based data retrieval and filtering

    Backend logic for managing job applications

    Form validation and data handling

🗄 Database Design

    Designed relational schema for users, companies, and job listings

    Implemented SQL queries for filtering, search functionality, and structured data retrieval

    Ensured data integrity through proper relationships and constraints
